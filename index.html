Currently, I'm working on building an off-grid chat application called "LoRa Chat" just for fun. It uses LoRa radio modules to create a decentralized mesh network where users can chat and transfer small amounts of data (sensor, offset/"wiki" etc.). If you are interested in checking it out, you can find it https://github.com/komppa/LoRaChat_v4

To make it easy to update the bootloader, partition table, firmware itself, and the web-based user interface for the ESP32, I created the LoRaChat Device Firmware Update application that uses WebUSB. This allows updating end nodes easily without the need to install SDKs and other software. Only web browser and one USB cable is required. You can check out that project here. It can also be accessed from the hosted site here.
